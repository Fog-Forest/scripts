# 📖 Scripts

🤪 随手写的一些乱七八糟脚本，持续更新中，觉得好用请点个⭐~

## 仓库简介

- **cloudthorn**
  无头浏览器 API 服务，类似于 Flaresolverr 的增强版，支持自定义请求头。
- **ddns-scripts**
  简易 DDNS 脚本，兼容 Cloudflare 服务商。
- **gcp-autostart**
  谷歌云抢占式实例状态实时监控及自动重启工具。
- **maccms-tool**
  MacCMS 视频信息整理工具。
- **mail-api**
  发送邮件接口，支持多种模板。
- **oracle-lookbusy**
  甲骨文云免费机器保活工具，采用智能资源占用策略。
- **passive-income**
  精选国内 IP 友好的挂机平台，助力实现闲置流量的变现。
